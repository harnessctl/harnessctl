# Task Clarification

Identify the missing core context needed to start the task without making blind assumptions.
Ask up to three concise questions to resolve these gaps, providing concrete options (e.g., A or B) instead of open-ended queries.
If further clarification is needed after the user replies, repeat this process.
Halt all execution and wait for the user's response before proceeding.
